# Markdown

## Code Block Indentation

Always indent fenced code blocks by 2 spaces and specify the language identifier.

OK:

  ```bash
  echo "hello"
  ```

NG:

```bash
echo "hello"
```

## Table Separator

Insert a space after the opening `|` and before the closing `|` in separator rows.

- OK: `| ------ | ------------- |`
- NG: `|------|-------------|`
